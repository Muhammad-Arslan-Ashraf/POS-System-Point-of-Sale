# 🍔 POS System — Point of Sale

A complete Point of Sale (POS) system built for restaurants and burger shops.
Manage categories, items, orders, and sales — all in one place.

---

## ✨ Features

- 📂 **Category Management** — Add, edit, delete categories (Burgers, Pizza, Drinks, etc.)
- 🍕 **Item Management** — Add items with image, price, and variations (Single/Size/Custom)
- 🛒 **Cart System** — Add to cart, increase/decrease quantity, remove items
- 🧾 **Order Types** — Dine In, Takeaway, Delivery
- 💳 **Payment Methods** — Cash & Online
- 🏷️ **Discount** — Apply percentage discount on orders
- 🖨️ **Receipt Printing** — Print receipt after every order
- 📊 **Sales Page** — View daily sales, order types breakdown
- 👨‍💼 **Cashier Info** — Cashier name and shift tracking
- 💾 **Local Storage** — All data saved locally, no backend required
